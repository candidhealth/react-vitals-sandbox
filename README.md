# React Vitals Sandbox

A sandbox app for exploring and testing components from [`@candidhealth/react-vitals`](https://www.npmjs.com/package/@candidhealth/react-vitals), Candid Health's React component library

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (ES2022+)
- [pnpm](https://pnpm.io/)

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Adding the library to your own app

```bash
npm install @candidhealth/react-vitals@alpha
```

## What's Inside

The sandbox demonstrates the full range of components from the library, including:

- **Typography** — Headlines, titles, body text, labels
- **Buttons** — Multiple intents, sizes, and variants
- **Form Controls** — InputBox, TextArea, Select, MultiSelect, Checkbox, Switch, RadioButtonGroup
- **Feedback** — Callouts, ErrorText, EmptyState
- **Overlays** — Modal, Tooltip, Menu
- **Layout** — Tabs, Collapsible sections, HeaderTile
- **Loading** — Spinner, LoadingState, LoadingBar, Skeleton
- **Utilities** — Copyable, Labels, Tags
- **Form Demo** — Full form example with `react-hook-form` and `zod` validation

Browse the full component library documentation at [components.joincandidhealth.com](https://components.joincandidhealth.com).
